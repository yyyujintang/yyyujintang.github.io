Network In Network 是发表于2014年ICLR的一篇paper。这篇文章采用较少参数就取得了Alexnet的效果，Alexnet参数大小为230M，而Network In Network仅为29M，这篇paper主要两大亮点：

从以上可以看出，MLP卷积层通过叠加”micro network”网络，提高非线性表达，而其中的”micro network”基本组成单元是1x1卷积网络。

1*1卷积的意义：

  1. 实现了不同通道同一位置的信息融合

  2. 可以实现通道数的降维或升维

    如果n小于之前通道数，则实现了降维，如果n大于之前通道数，则实现了升维。

用全局均值池化代替全连接层

该网络结构有三个MLP卷积层组成，每个MLP卷积层分别是一个普通卷积，加两个1x1卷积。以1000分类为例，最后一个1x1卷积输出的featuremap大小为6x6x1000。之后每个featuremap采用全局均值池化，输出1000个分类。由于没有全连接的大量参数，使用全局均值池化不需要参数，极大的降低了参数量。

Summary

Network In Network通过创新的创建MLP卷积层，提高了网络的非线性表达同时降低了参数量，用全局均值池化代替全连接层，极大的降低了参数量。
