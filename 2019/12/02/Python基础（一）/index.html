  • if语句后有:

3.4节编码规范 PEP8

  1. 每一个import语句只导入一个模块，尽量避免一次导入多个模块

  2. 不要在行尾添加分号，也不会要用分号把两条命令放在同一行

  3. 建议每行不超过80个字符（超过换行 ，1.加\再回车 2.小括号 两个或多个双引号）主要小括号要放在冒号之前

  4. 使用必要的空行来增加代码的可读性

  5. 运算符两侧（> < + - * /等），函数参数之间，逗号”,”两侧建议使用空格进行分割

  6. 避免在循环中使用+和+=运算符累加字符串（会创建不必要的即时对象）

  7. 适当使用异常处理结构 提高程序容错性

    3.5节命名规范

  • 模块名尽量短小，并且使用全部小写字母，可以使用下划线分隔多个字母game_register

  • 包名尽量短小，并且全部使用小写字母com.mr.book

  • 类名采用单词首字符大写形式（即Pascal风格）

  • 模块内部类由”“+Pascal风格的类名组成

  • 函数、类的属性和方法名全部采用小写字母，多个字母见用_分割

  • _常量命名时采用全部大写字母，可以使用下划线使用单下划线“_”开头的模块变量或者函数是受保护的

  • 使用双下划线“__”开头的实例变量或方法是类私有的

    4.1保留字与标识符

  • 保留字 区分大小写

  • import keyword

  • keyword.kwlist查看关键词列表标识符 名字

  • _标识符 表示保护变量

  • __标识符 表示类的私有成员

  • __ 标识符 _ 表示专用标识

    4.2变量

  • 内置type函数查看字符 类型

  • 内置id函数查看内存地址 在Python中多个变量指向同一个值时，它们共享同一个内存

  • 变量的命名1、必须是有效的标识符2.选择有意义的单词3.不能使用保留字4.谨慎使用l和o常量

    12.02

  • name.title(),name.upper(),name.lower(),name是变量，这几个函数表示将字符串转变为大写，大写和小写

  • python可以用+号来合并字符串制表符 \t,换行符\n删除空白 name.rstrip(),但如果要在变量中删除，必须赋值给变量，lstrip(),strip()分别表示删除开头和两端的空格整数运算，+-/,*表示乘方运算，可以用括号来指定运算
