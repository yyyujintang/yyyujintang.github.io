Mar 13, 2020 | 计算机视觉

感受野

在视觉神经系统中，视觉皮层中神经细胞的输出依赖于视网膜上 的光感受器。当光感受器受刺激兴奋时，会将神经冲动信号传导至视觉皮层

小卷积核通过多层叠加可取得与大卷积核同等规模的感受野

小卷积核优势

  • 小卷积核需多层叠加，加深了网络深度进而增强了网络容量
  • 增强网络容量的同时减少了参数个数

改造现有卷积操作的研究

  • 扩大原有卷积核在前层的感受野大小

    • 扩张卷积操作

  • 或使感受野不再是矩形区域而是更自由可变的形状

    • 可变卷积网络

分布式表示

词包模型

  • 源自自然语言处理领域

计算机视觉中

  • 将图像局部特征作为一个视觉单词
  • 将所有图像的局部特征作为词典
  • 一张图像可以用它的视觉单词描述
  • 视觉单词可以通过词典的形式映射成向量
  • 向量的每个维度可以对应一个明确的视觉模式或概念

深度卷积网络呈现分布式表示

  • 神经网络中的分布式表示指予以概念到神经元是多对多映射
  • 每个语义概念由许多分布在不同神经元中被激活的模式表示
  • 每个神经元又可以参与到许多不同语义概念的表示中去

即使是同一卷积核，在不同原图中响应的区域也可能大相径庭

神经网络响应的区域多呈现“稀疏”特性

  • 即响应区域集中且占原图比例较小

深度特征的层次性

浅层卷积神经网络学到的是基本模式

随着网络的加深，较深层除了泛化模式外，也开始出现了一些高层语义模式

得益于卷积网络特征的层次特性使得不同层特征可信息互补

对于单个网络模型而言，“多层特征融合”往往是一种直接且有效的网络集成技术，对于提高网络精度通常有较好表现

AlexNet

在卷积神经网络甚至连接注意机器学习发展进程中占据里程碑式的地位

首次将卷积神经网络用语ImageNet

  • 揭示CNN拥有强大的学习能力和表示能力
  • 海量数据也使卷积神经网络免于过拟合

使用GPU进行网格训练

  • 缩短研发周期与时间成本

一些训练技巧的引入

  • ReLU激活函数

  • 局部响应规范化操作

    • LRN
    • 要求对相同空间位置上相邻深度的卷积结果做规范化
    • 目前已作为各个深度学习工具箱的标准配置
    • k=0,n=N,α=1，β=0.5时是经典的l2规范化

  • 为防止过拟合采取的数据增广

    • data augmentation

  • 随机失活

    • dropout

VGG-Net

2014年ImageNet定位第一名，分类第二名的基础网络

广泛用于诸多问题

  • 物体候选框

    • object proposal

  • 细粒度图像定位于检索

    • fine0grained object localization and image retrieval

  • 图像协同定位

    • co-localization

使用技巧

  • 普遍使用小卷积核
  • 保持输入大小
  • 目的：确保在增加网络深度时确保各层输入大小随深度增加而不急剧减小
  • 网络卷积层通道数：3-64-128-256-512

Network-In-Network

用多层感知机（多层全连接层+非线性函数）替代简单的线性卷积层

一方面提供了网络层间映射的新可能

另一方面增加了网络卷积层的非线性能力，使上层特征可有更多复杂性与可能性的映射到下层

突破

  • 摒弃全连接层作为分类层的传统

  • 转而改用全局池化操作

    • global average pooling

  • 最后以池化结果映射到样本真实标记

ResNet

网络深度增加遇到的困难

  • 梯度弥散

    • 梯度过小会使回传的训练误差极其微弱

  • 梯度爆炸

    • 梯度太大会导致模型训练出现NaN现象

恒等映射

  • 高速公路网络
  • 不需要复杂的携带们和变换们去实现
  • 近路连接使梯度信息可以在多个神经网络层之间有效传播

残差学习模块

  • 左侧的残差函数
  • 右侧的对输入的恒等映射
  • 二者对应元素相加后再经过ReLU激活函数，从而形成整个残差学习模块
  • 多个残差学习模块堆叠而成的网络结构称作“残差网络”

类型

  • 常规残差模块

    • 由两个3*3堆叠而成

  • 瓶颈残差模块

    • bottleneck residual block
    • 3个，11,33,1*1
    • 1*1卷积起到降维或升维的目的
    • 提高计算效率

全局平均池化操作替代全连接层

  • 2015 GoogleNet
